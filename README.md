## 🌟 我们是谁

- **Miracle Coffee 灵感研究所** 是一个由咖啡爱好者、技术极客和创意玩家组成的小团队  
- 我们专注于三件事：
  1. **探索咖啡文化**：用味蕾记录生活的灵感瞬间  
  2. **打造共享咖啡体验**：办公室自助咖啡角 & 灵感激发的咖啡空间  
  3. **用技术记录生活**：GitHub 是我们灵感的记录本
